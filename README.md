# Art Institute Explorer

A minimalist React + TypeScript application for exploring the Art Institute of Chicago collection.

This first version focuses on the basic project setup, API integration, and a clean component structure.

## Tech Stack

- React + TypeScript (Vite)
- Tailwind CSS
- Zod
- Fetch API

## Getting Started

```bash
npm install
npm run dev
```

## Data Source

Art Institute of Chicago API
https://api.artic.edu
